Academic Student

This organiser website serves as a tool for all students. It contains multiple functions such as a To-Do list as well as a calendar for school tasks and assesments. To minimize procrastination, create a helathier school environment and a less stressful life for student, this website will aim to bring numerous benifits to all students. With a simplistic navigation bar the website will help you inspect the different webpages available. The "To-do" page will contain two lists, one for activities and the other for chores. The home page "AS" will contain a calendar in which you can organise your school tasks. Lastly, the "About" page contains a breif description about the website.

Enjoy!
